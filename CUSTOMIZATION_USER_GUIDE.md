# Customization Features User Guide

## Welcome to Customizations!

The Attendance Tracker app now includes powerful customization features that let you personalize the app's appearance to match your preferences and optimize battery life on AMOLED displays.

## Quick Start

### Accessing Customizations
1. Open the Attendance Tracker app
2. Tap the **Settings** tab at the bottom
3. Tap **Customizations** option
4. Explore and customize your app!

## Theme Modes

### System Default
- **What it does**: Follows your device's system theme settings
- **When to use**: If you want the app to automatically match your device theme
- **Battery impact**: Standard

### Light Theme
- **What it does**: Bright, high-contrast theme perfect for daytime use
- **When to use**: In bright environments or if you prefer light backgrounds
- **Battery impact**: Higher power consumption on OLED displays

### Dark Theme
- **What it does**: Dark gray theme that's easier on the eyes in low light
- **When to use**: Evening use or low-light environments
- **Battery impact**: Better than light theme on OLED displays

### AMOLED Theme âš¡ (Recommended for OLED/AMOLED displays)
- **What it does**: Uses pure black (#000000) backgrounds to turn off pixels completely
- **When to use**: 
  - If you have an OLED/AMOLED display (most modern smartphones)
  - When you want maximum battery savings
  - For the best contrast in dark environments
- **Battery impact**: **Best** - Can save 15-30% battery on OLED displays!
- **Visual features**:
  - True black backgrounds
  - Optimized text colors for readability
  - Reduced light emission

## Color Customization

This is the main accent color used throughout the app:

**Reset**: Tap the "Reset" button to return to the default blue

### Secondary Color
This provides accent variations and complementary highlights:
- Secondary buttons

**How to change**:
1. Tap the color circle next to "Secondary Color"
2. Choose from the color palette
3. Watch it update immediately!

**Reset**: Tap "Reset" to restore the default teal

## Troubleshooting

### Theme not applying?
- Try closing and reopening the app
- Ensure you've selected a theme mode
- Check if another app is forcing a theme

### Colors look off?
- Reset colors to defaults and try again
- Ensure your device screen settings aren't affecting colors
- Try a pre-built color scheme for guaranteed harmony

### Widget not updating?
- Long-press the widget and select "Refresh"
- Remove and re-add the widget
- Theme changes apply to widgets immediately in most cases

### Battery not improving with AMOLED?
- Ensure your device has an OLED/AMOLED display
- Check that AMOLED theme is actually selected
- Battery savings are most noticeable over time (hours/days)

## FAQ

**Q: Will my theme settings be saved?**
A: Yes! Your theme preference is saved in the app's database and persists across app restarts.

**Q: Can I use different themes at different times?**
A: Absolutely! You can change your theme anytime. Consider using System Default to automatically switch with your device.

**Q: Do custom colors work with all themes?**
A: Yes! Custom colors work with Light, Dark, and AMOLED themes.

**Q: How much battery does AMOLED theme really save?**
A: On OLED/AMOLED displays, AMOLED theme can save 15-30% battery compared to light themes, as black pixels are completely off.

**Q: What if I don't like my color choices?**
A: Simply tap "Reset" next to each color, or select the "Default" pre-built scheme to return to original colors.

**Q: Will future updates add more themes?**
A: We're always working on improvements! Future updates may include more themes, color schemes, and customization options.
